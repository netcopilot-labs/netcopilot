# Security Policy

## Reporting a vulnerability

If you find a security vulnerability in NetCopilot, please **do not open a public
issue**. Email **cg@netcopilot.io** with the details and we'll respond as quickly
as we can, work with you on a fix, and coordinate disclosure.

NetCopilot is **read-only on the network** by design — it never writes to devices,
which limits its blast radius. Please report anything that could expose
credentials, leak collected network data, or break that read-only invariant.

## Supported versions

The latest release is supported. Fixes land on the default branch and ship in the
next release.
